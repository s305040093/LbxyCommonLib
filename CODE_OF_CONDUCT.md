# 行为准则（Code of Conduct）

为建设开放、友好、专业的社区，本项目遵循类似 Contributor Covenant 的行为准则。

## 我们的承诺

作为维护者和贡献者，我们承诺：

- 尊重每一位参与者，不论其经验、性别、年龄、学历、种族或技术背景
- 在讨论中保持礼貌、克制和建设性
- 接受善意的批评意见，并以协作的方式解决分歧
- 将项目及社区的整体利益置于个人偏好之上

## 可接受的行为

- 使用友好、鼓励性的语言
- 尊重不同的技术路线和意见
- 主动分享知识与经验，帮助新人入门
- 在报告问题时提供尽可能完整的上下文与复现步骤

## 不可接受的行为

包括但不限于：

- 侮辱、贬损、恐吓或歧视性言行
- 人身攻击或恶意挑衅
- 发布与技术无关的广告、垃圾信息或恶意链接
- 在公共或私有交流中持续骚扰他人

## 维护者的责任

项目维护者有权：

- 删除、编辑或拒绝不符合本准则的评论、Issue、Pull Request 或其他贡献
- 在必要时暂时或永久地移除违反本准则的参与者

## 适用范围

本准则适用于：

- 本仓库内的所有协作空间（Issue、PR、Discussion 等）
- 与本项目相关的任何公开交流渠道

## 违规反馈

如发现违反本准则的行为，请通过提交 Issue 或直接联系维护者的方式进行反馈。  
维护者将在合理范围内对相关信息保密，并尽快处理。

## 致谢

本行为准则参考了 [Contributor Covenant](https://www.contributor-covenant.org)。
